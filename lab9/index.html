Lab 9 - Libraries and JQuery

Challenge

The challenge of this lab was to create HTML, CSS, and JS (with jQuery) files. We were also asked to create buttons that included event listeners.

Problems

At first, the only thing that showed up on my HTML file was the title. I consulted HTML files from former labs to see what I could change to make the sections appear. To do that, I had to add minor section classes. After that, I had to figure out why my buttons weren't showing up. The issue was that the IDs did not match. The biggest problem was that we couldn't figure out what the buttons were meant to do. We tried adding other features to make up for this, like adding an extra button that takes you to another website, mine goes to a song on Spotify. I also added some hover actions that change the color of the text space when you drag your mouse over it.

Results

The results were a webpage with some cool CSS and a button with a linked song in the output.
